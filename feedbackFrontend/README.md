# Feedback Collector Frontend

A React application for submitting and viewing feedback.

## Features

- Submit feedback with rating system
- View all submitted feedbacks
- Responsive design
- Real-time feedback submission

## Pages

- **Home** - Welcome page with navigation
- **Submit Feedback** - Form to submit new feedback
- **View Feedbacks** - Display all submitted feedbacks

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

Application runs on http://localhost:3000

## Note

Make sure the backend server is running on port 8002 for API calls to work.
